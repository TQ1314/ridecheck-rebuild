STEP 3 — SMS-FIRST FLOW
----------------------------------------------------

Update ONLY the send-payment-link API route.

Flow:

1. Load order
2. Generate payment link URL
3. Attempt SMS FIRST:

IF buyer_phone exists:
    try SMS
        success → mark channel='sms'
    catch error → fallback email

IF phone missing OR SMS fails:
    attempt email

IF email also unavailable:
    return error but DO NOT crash server
